# Pick Point

⏰ 개발기간
- `2025. 02. 01` ~ 

<br>

## 프로젝트 소개
화면을 터치하면 Point(점)들이 생성되고, 이 Point들 중 하나를 무작위로 선택하거나, 팀을 나누거나, 할 일을 배정할 수 있는 앱입니다.

1. Random Picker (랜덤 피커)
- 생성된 Point들 중 지정된 개수 만큼의 Point를 선택합니다.

2. Team Maker (팀 메이커)
- 생성된 Point들을 지정된 개수 만큼의 팀으로 묶어줍니다.

3. What To Do (랜덤 뽑기)
- 생성된 Point들을 미리 작성한 항목들에 배정해줍니다.
